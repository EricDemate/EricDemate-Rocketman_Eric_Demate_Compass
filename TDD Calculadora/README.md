# Introdução:
Aqui você encontra a descrição de como executar e informações sobre o projeto de uma calculadora com execução de TDD(Test Driven Development).<br/><br>

Após instalar o Mocha e Chai atraves do terminal integrado do Vs Code,devemos criar uma pasta chamada **"src"**, onde será criado nosso arquivo **"Calculadora.js"**<br>

![](https://user-images.githubusercontent.com/64226050/183100436-e3f9c13a-d7c0-4b8d-a6b7-3e3bb68de32c.png)

E para montar a rotina de testes devemos criar uma pasta chamada **"test"**, onde será criado nosso arquivo **"Calculadora.spec.js"**.<br><br>

![](https://user-images.githubusercontent.com/64226050/183100918-326a9fef-e9e7-40b0-8714-b630f591d0d6.png)

No arquivo de **"Calculadora.spec.js"** não podemos esquecer de importar as bibliotecas de assert, chai e Calculadora. <br><br>

# Executar o teste

Para realizar o teste primeiramente temos que nos certificar se estamos com o terminal na pasta **"Mocha_Chai"**.

Após isso podemos executar o comando:

```
npm test
```
E para finalizar como mostra na imagem abaixo será exibido o teste sendo realizado.

![](https://user-images.githubusercontent.com/64226050/183115501-af0f391a-e0d6-4afa-b734-ffe3663ba30c.png)
